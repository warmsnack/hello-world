## Cursor Cloud specific instructions

This repository is the classic GitHub "hello-world" onboarding repo. It contains only
`README.md` — there is **no application, no dependencies, no build system, no tests, and no
services** to run.

- There is nothing to install. The environment update/setup step is a no-op.
- There is no lint/test/build/run command. Do not spend time searching for a package
  manifest, lockfile, or dev server — none exist.
- The repo's only "functionality" is the Git/GitHub workflow itself (edit a file, commit,
  push, open a PR). Verify the environment by making a small edit and completing that
  workflow.

If real application code is added later, replace this section with actual setup, dependency,
and run/test/build instructions.
